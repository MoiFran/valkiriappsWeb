## [0.1.0] - 2025-01-22

### Añadido

- Configuración inicial del proyecto con Next.js 15 y React 19
- Componente Hero con efectos WebGL y animaciones GSAP
- Sección Misión con valores corporativos
- Sección Servicios con 4 servicios principales
- NavBar con navegación gooey y efectos de partículas
- Efectos visuales personalizados: LaserFlow y ElectricBorder
- Sistema de smooth scroll con Lenis
- Integración de Three.js y React Three Fiber
- Componentes de animación con Framer Motion
- Sistema de diseño con CSS Modules y variables
- Configuración TypeScript completa
- Hooks personalizados: useSmoothScroll, useWindowSize
- Constantes centralizadas para company info, services, navigation
- Tipos TypeScript para todas las interfaces
- Estructura de carpetas profesional y escalable

### Configuración

- ESLint con reglas profesionales
- Prettier para formato de código
- Husky + lint-staged para pre-commit hooks
- Variables de entorno (.env.example)
- EditorConfig para consistencia
- Scripts npm útiles (format, type-check, lint:fix)

### Documentación

- README.md completo con instrucciones
- DOCS.md con documentación de componentes
- CONTRIBUTING.md con guías de contribución
- CHANGELOG.md (este archivo)
- LICENSE MIT

### Performance

- Optimización de imágenes con next/image
- Code splitting automático de Next.js
- Lazy loading de componentes pesados
- CSS optimizado con modules

### Accesibilidad

- Semántica HTML correcta
- Soporte de navegación por teclado
- ARIA labels en componentes interactivos
- Contraste de colores accesible

## Tipos de Cambios

- `Añadido` - Para nuevas características
- `Cambiado` - Para cambios en funcionalidad existente
- `Deprecado` - Para características que se eliminarán
- `Eliminado` - Para características eliminadas
- `Corregido` - Para corrección de bugs
- `Seguridad` - Para vulnerabilidades
